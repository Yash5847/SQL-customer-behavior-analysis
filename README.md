This project focuses on analyzing customer purchase behavior using SQL.

## Problem Statement
Find customers who placed orders in February but did not place any order in March.

## Business Use Case
Identifying customers who stop purchasing helps businesses design re-engagement and retention strategies.

## Dataset Tables
- customers (customer_id, customer_name)
- orders (order_id, customer_id, order_date)

## SQL Concepts Used
- JOIN
- Subqueries
- Date functions
- Filtering conditions

## Key Learning
This exercise strengthened my ability to write clean SQL queries for month-based customer behavior analysis.
